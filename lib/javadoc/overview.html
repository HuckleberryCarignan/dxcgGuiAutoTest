DxCG Java API

Here is an example of how the DxCG API can be used.
```java import com.veriskhealth.dxcg.api.DxCGAPI; /** * Class to help demonstrate the functionality of the API. * */ public class DxCGDemo { private DxCGDemo() { } public static void main(String[] args) throws Exception { DxCGAPI dxcgApi = new DxCGAPI(); String configFilePath = "C:/temp/model26.cfg"; dxcgApi.initialize(configFilePath); int returnValue = dxcgApi.processMembers(configFilePath); } } ```
